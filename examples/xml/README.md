# XML output

This example demonstrates how a JavaScript object can be used to create
an XML document.

The output of this script is:

```xml
<?xml version="1.0"?>
<person>
  <name>Eric</name>
  <age>69</age>
  <height>175</height>
  <skills>JavaScript</skills>
  <skills>C</skills>
  <skills>C++</skills>
  <skills>Java</skills>
  <skills>PHP</skills>
  <skills>ARM Assembler</skills>
  <address>
    <house>69</house>
    <street>Acacia Avenue</street>
    <city>Anywhere</city>
    <Country>UK</Country>
  </address>
  <address>
    <house>221b</house>
    <street>Baker Street</street>
    <city>Anywhere</city>
    <Country>UK</Country>
  </address>
</person>
```
